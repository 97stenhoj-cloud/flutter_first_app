## 2. DESCRIPTION OF SERVICE

Connect is a conversation game application designed to facilitate meaningful discussions and strengthen relationships through curated questions. The App provides structured conversation experiences for different relationship types and social settings.

### 2.1 Core Features

The App offers the following game modes:
- **Couple Mode:** Intimate questions designed for romantic partners
- **Friends Mode:** Engaging topics for friend groups
- **Family Mode:** Wholesome conversations suitable for family settings
- **Personal Mode:** Custom question deck creation (Premium subscribers only)
- **Pandora Mode:** Real-time multiplayer collaborative game sessions

### 2.2 Question Library

- **Free Tier:** Access to 3 categories per mode, with 30 questions per category
- **Premium Tiers:** Access to 24+ premium categories, with 75 questions per category
- **Question Categories:** Various topics including relationships, personal growth, preferences, experiences, and more

### 2.3 Additional Features

- **Favorites System:** Save preferred questions for quick access
- **Custom Decks:** Create personalized question collections (Premium feature)
- **Spark Mode:** AI-powered personalized question generation (Premium/Premium+ feature)
- **Multi-language Support:** Available in 13 languages (English, Danish, German, Spanish, French, Italian, Norwegian, Dutch, Polish, Portuguese, Romanian, Swedish, Finnish)
- **Offline Mode:** Limited gameplay functionality without internet connection
- **Theme Customization:** Dark and light mode options
